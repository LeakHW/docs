# UI Profiles

This guide will go over everything there is to know about UI profiles on Leak.

---

## What are UI profiles?
UI Profiles are a certain set of styles that can be applied to a website. For example, they can change the layout of the website to make it more accessibility-friendly.

---

## Accessing the UI profiles menu
To access the UI profiles menu, open the Leak menu and press the second icon at the bottom (the palette icon).

---

## Suggesting tools
To suggest UI profiles to be added to Leak, click [here](https://github.com/LeakHW/Extension/issues/new?template=feature_request.yml) to submit a feature request.
