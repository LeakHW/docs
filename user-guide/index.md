### **Core Features**

Leak has 3 main features:

- **Tools**: Tools are pieces of code designed to do a certain thing. For example, open a calculator.
- **UI Profiles**: UI Profiles are a certain set of styles that can be applied to a website. For example, they can change the layout of the website to make it more accessibility-friendly
