# The Menu

This page will go over the Leak Menu.

---

## What is the Leak Menu?
The Leak menu is the main menu for Leak. It can be accessed two ways:
- Pressing the Leak button built into the website (if its supported)
- Pressing the extension icon

---

## What are the buttons on the Leak Menu?
At the bottom of the Leak menu, there are 4 icons:
- [Tools](tools) - The default tab on the menu
- [UI Profiles](uiprofiles) - The menu for selecting different UI profiles
- About (Undocumented) - About the extension
- Settings (Undocumented) - Settings for the extension including optional opt-ins
Click on one of these to learn more about them.
