# Leak Tools

This guide will go over everything there is to know about tools on Leak.

---

## What are tools?
Tools are pieces of code designed to do a certain thing. For example, open a calculator.

---

## Accessing tools
The tools page is the default page of the Leak menu. To get back to it after going to a different page, press the first icon at the bottom of the Leak Menu.

---

## Suggesting tools
To suggest tools to be added to Leak, click [here](https://github.com/LeakHW/Extension/issues/new?template=feature_request.yml) to submit a feature request.
