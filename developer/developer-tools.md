# Developer tools

In the Leak Menu, there is a hidden developer section inside **Settings**. This guide will go over it.

---

## Going to the menu
Open the Leak Menu, go to **Settings**, then open the **About** category and click the version number until you get a popup. Developer Mode will then be enabled.

Once enabled, a new **Dev Tools** category will appear in the Settings sidebar.

---

## Options in the menu
- Clear all storage - Resets all settings and cached data
- Developer mode - Toggle the Dev Tools settings category on or off
- Global debug logging - Verbose logging across all tools
- Experimental tools - Toggle if experimental tools are shown
- Show Example Tool - Toggle the example/template tool

If any registered tool is marked as `devOnly`, it may also appear under the Dev Tools category when Developer Mode is enabled.
