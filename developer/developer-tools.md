# Developer tools

In the Leak Menu, there is a hidden developer menu. This guide will go over this menu.

---

## Going to the menu
To make the menu visible, go to the about tab and click the version number until you get a popup. Developer tools are now enabled. There should now be a new icon that looks like `</>`.

---

## Options in the menu
- Clear all storage - Resets all settings and cached data
- Developer mode - Toggle developer mode tab off or on
- Global debug logging - Verbose logging across all tools
- Experimental tools - Toggle if experimental tools are shown
